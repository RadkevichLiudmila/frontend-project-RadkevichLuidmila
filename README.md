# frontend-project-RadkevichLuidmila

###################################
Сайт написан с помощью SPA. Содержит 4 страницы.

###################################
На первой странице можно выбрать цвет игрока, щелкая по цветным кружочкам.
Код реализуется с помощью jQuery.

###################################
На второй странице написаны правила игры.

###################################
На третей странице 2d игра. После ее запуска шарик, т е игрок завязывается на курсор, сам курсор исчезает и человек управляет игроком с помощью компьютерной мышки, либо пальца.
Каждые 3 секунды на поле появляется клад, который нужно словить и препятствие, до которого нельзя дотрагиваться. Если игрок поймал клад раньше чем прошло 3 секунды, то клад исчезает,
а вместо него появляется новый бонус в другом месте. За каждый пойманный бонус игроку начисляются 10 очков.
Препятствие же не пропадает с поля, каждый 3 секунды их становится все больше и больше.
Цель набрать как можно больше очков.
Игра заканчивается, когда игрок ударяется о стену или о препятствие.

# Игра построена с помощью DOM-элементов, обработаны события mouse и touch.

# Когда игра останавливается, выскакивает модальное окно, чтобы узнать нужно ли сохранить результаты. Результаты сохраняются на сервер и в локальное хранилище.
Это нужно для того, чтобы потом можно было вывести списком лучших игроков и лучший результат с личного устройства.
# Модальное окно реализованно с помощью jQuery.

# Во время сохранения результатов в окне может появится прокрутка ожидания. Это происходит, когда низкая скорость соединения с интернетом.
# Прокрутка нарисована на canvas. Движение завязано на таймер, прокручивается с помощью CSS свойства transform.

#####################################
 На последней странице выводится топ 10 лучших игроков, и лучший результат с личного компьютера.
